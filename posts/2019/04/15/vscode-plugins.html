# Mes extensions préférées de VSCode

Cela fait maintenant un petit moment que j’utilise VSCode aussi bien au boulot qu’en perso. Je trouve cet IDE vraiment très agréable à utiliser et aussi très performant.

Mais il possède aussi une autre force: les extensions.

Voici une liste non exhaustives des extensions de VSCode que j’utilise le plus souvent.

# 1. Bracket Pair Colorizer

Bracket Pair Colorizer permet d’appairer les parenthèses, accolades et autres crochets par des couleurs pour mieux identifier leur fin.

Lien: https://marketplace.visualstudio.com/items?itemName=CoenraadS.bracket-pair-colorizer

# 2. Code Time

Cette extension permet de visualiser le temps passé à coder ainsi que de récupérer différents métriques tel que : le nombre de lignes ajoutées, le nombre de lignes supprimées, les caractères ajoutés, les caractères supprimés, les heures où vous avez le plus codé…

Après avoir installé cette extension, il ne vous restera qu’à créer un compte sur le site d’appsoftware.com et d’autoriser le lien avec l’extension.

# 3. Auto Rename Tag / Auto Close Tag

Vous est-il déjà arrivé d’oublier de fermer un tag HTML ou de devoir en renommer un en oubliant de le faire pour le tag de fin du bloc?

C’est ce que permettent ces deux extensions. Lorsque vous devez renommer un tag, vous renommez le tag de début et le tag de fin sera renommé automatiquement. De la même façon, lorsque vous déclarez le tag de début d’un bloc, le tag de fin de bloc est inséré automatiquement. Grâce à ces deux extensions, vous aurez moins d’erreurs dans votre code et vous gagnerez du temps dans le développement des fonctionnalités de vos pages HTML.

# 4. Formatting Toggle

Parfois, lorsque vous êtes en pleine session de code et que vous sauvegardez vos modifications dans votre code, vous ne souhaitez pas que les formatter tels que Prettier ou Beautify fassent une passe sur ce code en cours de développement.

C’est ce que permet cette extension en activant ou désactivant les formatter tout simplement.

Voici un petit aperçu des extensions principales que j’utilise avec VSCode. Je complèterai surement cette liste au fur et à mesure des nouvelles extensions que j’ajouterai à mes environnements de développement.
